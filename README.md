# Լեյտենանտը

Հոգեբանական serious narrative web game՝ գրված React + Vite + TypeScript-ով։

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Նախագիծը չունի backend, database, Tailwind, Redux կամ արտաքին UI library։ Կարելի է deploy անել Vercel կամ Netlify-ում։
